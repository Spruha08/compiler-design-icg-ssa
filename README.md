# Compiler Design - Recent Methodologies in Intermediate Code Generation (ICG)
This project demonstrates important concepts used in Intermediate Code Generation (ICG) in Compiler Design.
The implementation includes:
- Three Address Code (3AC) Generator
- Static Single Assignment (SSA) Simulator
Both implementations are built using Python and Streamlit.
---
# Features:-
## Three Address Code (3AC):
- Converts infix expressions into postfix expressions
- Generates intermediate three address code
- Uses temporary variables (t1, t2, etc.)
## Static Single Assignment (SSA):
- Converts statements into SSA form
- Creates unique variable versions
- Demonstrates modern compiler optimization concepts
